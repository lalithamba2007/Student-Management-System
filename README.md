# Student Management System (Java)

A console-based Student Management System built using Java.
Implements CRUD operations with file-based persistence.


## Features

* Add Student
* View All Students
* Update Student
* Delete Student
* Search by ID
* Search by Name
* Search by Course
* Persistent storage using file handling


## Tech Stack

* Java
* Collections Framework (`Map`, `List`, `TreeMap`)
* File Handling (`BufferedReader`, `BufferedWriter`)
* OOP Principles

## How to Run

javac Main.java
java Main
